A simple Go modules guide

Create a module:

$ mkdir /home/gopher/mod1
$ cd /home/gopher/mod1
$ git init
Initialized empty Git repository in /home/gopher/mod1/.git/
$ git remote add origin https://gopher.live/{{{.REPO1}}}.git
$ go mod init gopher.live/{{{.REPO1}}}
go: creating new go.mod: module gopher.live/{{{.REPO1}}}

Write a readme:

## `gopher.live/{{{.REPO1}}}`

Write a readme:

package main

import "fmt"

func main() {
	fmt.Println("Hello, world!")
}

Commit and push:

$ git add -A
$ git commit -m "Initial commit"
[main (root-commit) abcd123] Initial commit
 3 files changed, 11 insertions(+)
 create mode 100644 README.md
 create mode 100644 go.mod
 create mode 100644 main.go
$ git push -u origin main
remote: . Processing 1 references        
remote: Processed 1 references in total        
To https://gopher.live/{{{.REPO1}}}.git
 * [new branch]      main -> main
Branch 'main' set up to track remote branch 'main' from 'origin'.

Use the module:

$ mkdir /home/gopher/mod2
$ cd /home/gopher/mod2
$ go mod init mod.com
go: creating new go.mod: module mod.com
$ go get gopher.live/{{{.REPO1}}}
go: downloading gopher.live/{{{.REPO1}}} v0.0.0-20060102150405-abcde12345
go: gopher.live/{{{.REPO1}}} upgrade => v0.0.0-20060102150405-abcde12345
$ go run gopher.live/{{{.REPO1}}}
Hello, world!
